# 문제 정의

오늘의집에서 상품을 판매하는 중소 브랜드(커튼, 린넨, 생활잡화 등)는 하루 30~100건 이상의 반복적인 고객 문의(치수 계산, 수량 안내, 설치 가능 여부 등)를 받고 있습니다. 이 문의들의 대부분은 유사한 패턴의 질문(가로/세로 치수, 장수, 레일 여부, 암막 여부 등)이지만, 오늘의집은 공식 API가 없어 자동 응답 기능을 구현할 수 없습니다.

고객들은 상품 상세 가이드를 보더라도 자신의 상황에 직접 대입하여 계산하기 어려워하며, CS 대응 속도가 오늘의집 노출 순위와 매출에 직접적인 영향을 미치기 때문에 판매자들은 문의 답변에 매일 수 시간을 소비하고 있습니다. 이로 인해 별도 CS 인력 채용을 고려하지만 중소 브랜드에게는 인건비 부담이 큽니다.

# 타겟 사용자 및 사용 사례

## 핵심 사용자

1. **오늘의집에만 입점한 중소 브랜드 대표/운영자**
   - 1인~3인 사업체
   - CS 자동화에 관심이 높음
   - 시간 절약에 매우 민감

2. **커튼/침구/생활 가구 카테고리 판매자**
   - 정형화된 규칙이 있는 제품군
   - 반복적인 CS 문의가 매우 많음

3. **상품 상세 가이드가 있음에도 '대입 계산이 어려운' 문제를 가진 고객층을 대상으로 하는 브랜드**

## 사용 사례

### 사례 1: 아침 출근 전 문의 처리
브랜드 대표가 아침에 일어나 오늘의집 Q&A를 확인하니 밤사이 15건의 문의가 쌓여있습니다. 각 문의를 복사하여 AI 상담 에이전트에 붙여넣으면 3~5초 내로 브랜드 톤에 맞춘 정확한 답변 초안을 받고, 검토 후 바로 오늘의집에 등록합니다. 기존 30~40분 소요되던 작업이 5분 내로 완료됩니다.

### 사례 2: 복잡한 치수 계산 문의
"가로 210에 세로 230인데 몇 장 주문해야 하나요? 이음선 생기나요?"라는 문의가 들어왔습니다. 판매자는 문의 내용을 AI에게 전달하고, AI는 규칙 엔진으로 실측 가로 × 1.5배 계산, 패널 조합 계산, 이음선 발생 여부를 자동으로 판단하여 "안녕하세요, 맞춤제작 커튼 전문업체 데코지오입니다. 고객님의 경우 350폭 3장을 추천드리며, 3장 이상이므로 이음선이 발생할 수 있습니다..."와 같은 완성형 답변을 제공합니다.

### 사례 3: 브랜드 일관성 유지
여러 직원이 CS를 담당할 경우 답변 톤이 제각각일 수 있습니다. AI 상담 에이전트는 브랜드별로 설정된 톤("안녕하세요, 맞춤제작 커튼 전문업체 데코지오입니다. 😊")을 항상 유지하여 브랜드 이미지를 일관되게 관리할 수 있습니다.

# 제안 해결책

오늘의집 판매자가 고객 문의를 복사하여 AI에게 전달하면, **브랜드 맞춤형 AI 상담 에이전트**가 3~5초 내로 브랜드 톤에 맞춘 정확한 답변 초안을 생성해주는 B2B SaaS를 제공합니다.

이 솔루션은 다음 3가지 핵심 요소로 구성됩니다:

1. **규칙 엔진**: 브랜드별 치수 계산, 수량 조합, 이음선 발생 여부 등을 자동으로 계산
2. **File Search RAG**: 브랜드의 공식 정책 문서, Q&A 패턴, 톤앤매너를 학습하여 정확한 답변 생성
3. **반자동 시스템**: 완전 자동이 아닌 '검토 후 붙여넣기' 방식으로 잘못된 답변 업로드 리스크 제거

별도 CS 인력 채용 없이 월 3~10만 원 수준의 구독료로 CS 대응 시간을 70% 이상 절감할 수 있습니다.

# 목표 및 성공 지표

## 목표

오늘의집 판매자의 반복적인 CS 업무를 AI로 자동화하여 시간을 절약하고, 빠르고 정확한 답변으로 고객 만족도와 매출을 향상시킨다.

## 성공 지표

| 지표 | 목표값 | 측정 방법 |
|------|--------|-----------|
| 답변 생성 시간 | 7초 이하 | API 응답 시간 측정 |
| CS 대응 시간 절감율 | 70% 이상 | 사용 전후 시간 비교 설문 |
| 초기 판매자 확보 | 5개 브랜드 | 1개월 내 유료 전환 |
| 월 구독 전환율 | 30% 이상 | 무료 체험 → 유료 전환율 |
| 답변 정확도 (수정률) | 수정 10% 이하 | 생성 답변 중 수정 필요 비율 |
| 브랜드당 월간 처리 문의 수 | 평균 200건 이상 | 사용 로그 분석 |

# 경쟁사 분석

## 기존 CS 자동화 솔루션

### 1. 챗봇 플랫폼 (채널톡, 카카오톡 상담톡 등)
- **해결 방식**: 웹사이트/앱에 챗봇 위젯 설치, 시나리오 기반 자동 응답
- **한계**:
  - 오늘의집은 공식 API 미제공으로 챗봇 연동 불가
  - 시나리오 작성에 많은 시간 필요
  - 복잡한 치수 계산 등 규칙 기반 응답 처리 어려움
- **차별점**: 우리는 오늘의집 Q&A에 특화되어 있으며, 복사-붙여넣기만으로 사용 가능

### 2. AI 챗봇 SaaS (Zendesk AI, Intercom 등)
- **해결 방식**: AI 기반 자동 응답, 티켓 관리 시스템
- **한계**:
  - 글로벌 서비스로 한국어 및 특수 도메인(커튼 치수 계산 등) 대응 약함
  - 월 수십만 원 이상의 높은 비용
  - 오늘의집 플랫폼 직접 연동 불가
- **차별점**: 우리는 한국 중소 브랜드 대상, 월 3~10만 원 수준의 저렴한 가격, 브랜드별 맞춤형 규칙 엔진 제공

### 3. GPT 기반 직접 활용
- **해결 방식**: ChatGPT에 문의 내용 복사하여 답변 요청
- **한계**:
  - 브랜드 톤, 정책, 규칙을 매번 프롬프트에 입력해야 함
  - 치수 계산 등 정확한 수치 처리 어려움
  - 답변 일관성 부족
- **차별점**: 우리는 브랜드별 File Search로 정책 문서 자동 참조, 규칙 엔진으로 정확한 계산, 일관된 톤 유지

## 우리 서비스의 핵심 차별점

1. **오늘의집 판매자 특화**: API 없는 오늘의집 환경에 최적화된 '복사-붙여넣기' 워크플로우
2. **브랜드 맞춤형 규칙 엔진**: 커튼 치수 계산 등 도메인 특화 로직 자동 처리
3. **반자동 시스템**: 완전 자동이 아닌 검토 후 등록 방식으로 안정성과 통제력 확보
4. **저렴한 가격**: 월 3~10만 원으로 중소 브랜드도 부담 없이 사용 가능
5. **빠른 도입**: 문서 업로드만으로 1시간 내 사용 시작 가능

# MVP 요구사항

## Phase 1: 핵심 기능 (1~2주 목표)

### 1. 브랜드 온보딩
- 브랜드 기본 정보 등록 (브랜드명, 대표 인사말, 마무리 멘트)
- File Search Store 자동 생성
- 정책 문서 업로드 (policy.txt, qna.txt, tone.txt)

### 2. AI 답변 생성 API
- 고객 문의 입력 (텍스트 복사-붙여넣기)
- 선택적 치수 정보 입력 (가로/세로/설치 타입)
- 규칙 엔진 실행하여 패널 조합, 이음선 여부 계산
- File Search로 브랜드 문서 검색
- Gemini 모델로 브랜드 톤에 맞춘 답변 생성
- 생성된 답변 텍스트 출력 (복사 버튼 제공)

### 3. 규칙 엔진 (커튼 도메인)
- 실측 가로 × 1.5배 → 목표 가로폭 계산
- 70/140/280/350 단위로 패널 조합 계산
- 장수 3장 이상 → 이음선 경고
- 아일렛/핀형/박스 유형별 높이 계산
- 박스 깊이 기준 레일 추천

### 4. 간단한 Web UI
- 문의 입력 폼
- 답변 결과 표시 및 복사 기능
- 브랜드 선택 드롭다운 (향후 멀티 브랜드 대응)

## Phase 2: 관리 기능 (3~4주 목표)

### 5. 브랜드 문서 관리
- Web Admin에서 문서 업로드/수정/삭제
- File Search Store 자동 갱신
- 문서 버전 관리

### 6. 사용 로그 및 분석
- 처리된 문의 내역 저장
- 답변 수정 여부 트래킹
- 일별/주별 사용 통계

### 7. 답변 품질 개선
- 생성된 답변에 대한 피드백 수집 (좋음/나쁨)
- 수정된 답변 저장하여 학습 데이터로 활용
- 자주 수정되는 패턴 분석

## Phase 3: 확장 기능 (향후)

### 8. 멀티 브랜드 지원
- 브랜드별 독립적인 File Search Store
- 브랜드별 구독 관리
- 브랜드별 사용량 제한

### 9. 고도화 기능
- 이미지에서 치수 자동 추출 (OCR)
- 문의 내용에서 치수 자동 파싱
- 자주 묻는 질문(FAQ) 자동 생성
- 오늘의집 Q&A 크롤링 자동화 (법적 검토 후)

## 핵심 사용자 플로우 (MVP)

1. **판매자 온보딩**
   - 브랜드 정보 입력
   - 정책 문서 3개 업로드 (policy.txt, qna.txt, tone.txt)
   - 온보딩 완료 (약 10분 소요)

2. **일상적인 문의 처리**
   - 오늘의집 Q&A에서 고객 문의 복사
   - AI 상담 에이전트 웹페이지 접속
   - 문의 내용 붙여넣기
   - (필요시) 치수 정보 입력
   - "답변 생성" 버튼 클릭
   - 3~5초 대기
   - 생성된 답변 검토
   - "복사" 버튼 클릭
   - 오늘의집 Q&A에 답변 붙여넣기 및 등록

3. **문서 업데이트**
   - 정책 변경 시 Web Admin 접속
   - 해당 문서 수정 또는 재업로드
   - 자동으로 File Search Store 갱신
   - 즉시 새로운 정책 반영
