🧩 4. 기능 설명 섹션 (한 섹션=한 기능)

각 기능은 “한 문장 가치 + 짧은 설명”으로 구성한다.

🔹 기능 1: 장수/높이 자동 계산 엔진

가로/세로/스타일을 넣으면 규칙 엔진이 자동 계산.

140/280/350 단위 조합, 이음선 여부까지 자동 판단.

🔹 기능 2: 브랜드 톤 복제

인사말/마무리/문체를 너가 올린 문서 그대로 반영.

“사장님이 말하는 톤 그대로” 답변 생성.

🔹 기능 3: 정책 문서 기반 정확한 답변 (File Search)

브랜드의 가이드, Q&A 문서 자동 청크 → RAG 구동.

고객이 뭐라고 물어도 브랜드 문서를 근거로 답변함.

🔹 기능 4: 반자동 구조로 안전

AI가 답변을 초안 생성 →

판매자가 검토 후 오늘의집에 붙여넣기
